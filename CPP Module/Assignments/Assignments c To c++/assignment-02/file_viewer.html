🧮 Simple Calculator Program

C++ Program for Basic Arithmetic Operations

Source Code

#include <iostream>
using namespace std;
int main()
{
    int num1;
    cout << "Enter the number-1: ";
    cin >> num1;
    int num2;
    cout << "Enter the number-2: ";
    cin >> num2;
    char op;
    cout << "Enter the operator: ";
    cin >> op;
    if (op == '+')
    {
        cout << "Addition: " << num1 + num2;
    }
    else if (op == '-')
    {
        cout << "substraction: " << num1 - num2;
    }
    else if (op == '*')
    {
        cout << "multiplication: " << num1 * num2;
    }
    else if (op == '/')
    {
        cout << "division: " << num1 / num2;
    }
    else if (op == '%')
    {
        cout << "modulus: " << num1 % num2;
    }
    else
    {
        cout<<"Enter a valid input";
    }
    return 0;
}

Program Explanation

1
Headers & Namespace
Program को input/output के लिए iostream की जरूरत है। std namespace का use करके code को छोटा लिखते हैं।
#include <iostream>
using namespace std;
2
Input लेना
User से 2 numbers लिए जाते हैं और एक operator। cout से message दिखाते हैं, cin से input लेते हैं।
cin >> num1; // user का input लेता है
cout << "msg"; // message print करता है
3
Operator Check करना
if-else if statements का use करके कौन सा operation करना है यह check किया जाता है।
if (op == '+') { ... }
else if (op == '-') { ... }
4
Result Calculate करना
के अनुसार operation perform करके result को print किया जाता है।
Addition: 5 + 3 = 8
Subtraction: 5 - 3 = 2

🔄 Program Flow

User से पहला number input लेता है
User से दूसरा number input लेता है
User से operator (+, -, *, /, %) input लेता है
Operator को check करके operation perform करता है
Result को print करता है
